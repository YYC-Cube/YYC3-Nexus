**所有5个失败文件均为同一根因：导入路径层级错误**

| 目录层级                   | 正确路径            | 错误路径            |
| -------------------------- | ------------------- | ------------------- |
| `tests/components/ui/`     | `../../../src/app/` | `../../src/app/`    |
| `tests/components/panels/` | `../../../src/app/` | `../../src/app/`    |
| `tests/components/pages/`  | `../../../src/app/` | `../../src/app/`    |
| `tests/components/`        | `../../src/app/`    | `../../../src/app/` |

**修复方案：批量修正5个文件的导入路径，预计影响 ~5行/文件。**

| 经验           | 说明                                            |
| -------------- | ----------------------------------------------- |
| Portal渲染     | 内容不在container中，需用document.querySelector |
| waitFor来源    | 必须从 @testing-library/react 导入，非vitest    |
| pointer-events | 使用fireEvent.pointerDown替代mouseDown          |
| aria-disabled  | Radix使用aria-disabled替代原生disabled          |
| CheckboxItem   | 需onSelect阻止菜单关闭                          |
| PanelGroup     | Panel必须在PanelGroup内部渲染                   |

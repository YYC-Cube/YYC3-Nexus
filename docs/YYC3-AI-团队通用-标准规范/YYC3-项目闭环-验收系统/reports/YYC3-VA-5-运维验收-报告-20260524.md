## 五、经验教训

### 5.1 技术经验

| 经验 | 详情 |
|------|------|
| Radix UI测试 | Portal渲染需要waitFor和document.querySelector，不能依赖container |
| Vitest waitFor | 必须从@testing-library/react导入，不能从vitest导入 |
| 导入路径层级 | tests/components/ui/需要../../../src/，tests/components/需要../../src/ |
| Tailwind CSS类名 | 断言需匹配实际编译输出（如flex而非inline-flex） |
| react-resizable-panels | Panel必须在PanelGroup内，且需必填direction属性 |

### 5.2 流程经验

| 经验 | 详情 |
|------|------|
| 五维驱动落地 | 五高→架构，五标→规范，五化→流程，五维→评估 |
| 文档闭环 | YAML Front Matter + 品牌标识 + 版本追溯 + 模版引擎 |
| 验收标准化 | 6阶段提示词系统使验收可重复、可追溯 |

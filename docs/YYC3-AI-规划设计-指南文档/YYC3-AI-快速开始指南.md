# YYC³ 设置系统 - 快速开始指南

## 🚀 快速开始

### 1. 访问设置页面

在 YYC³ CloudPivot Intelli-Matrix 应用中，点击侧边栏的"设置"按钮（⚙️ 图标）即可进入设置页面。

### 2. 导航结构

设置页面分为 **10 个主要模块**：

| 模块 | 功能 | 快捷键 |
|------|------|--------|
| 📱 账号信息 | 管理个人信息和头像 | - |
| ⚙️ 通用设置 | 主题、语言、编辑器配置 | - |
| 🤖 智能体管理 | AI 智能体配置 | - |
| 🔌 MCP 连接 | 模型上下文协议管理 | - |
| 💻 模型配置 | AI 模型和 API 密钥 | - |
| 📂 上下文管理 | 代码索引和文档集 | - |
| 💬 对话流设置 | 对话行为和通知 | - |
| 📜 规则管理 | 自定义规则和约束 | - |
| ⚡ 技能管理 | 自定义技能和能力 | - |
| 📥 导入/导出 | 配置备份和迁移 | - |

### 3. 全局搜索

在页面顶部的搜索框中输入关键词，即可快速找到相关设置项。支持：

- 设置项名称
- 描述内容
- 配置值
- 智能体/规则/技能名称

## 📋 常见任务

### 1. 创建自定义智能体

1. 进入"智能体管理"模块
2. 点击"创建智能体"按钮
3. 填写智能体信息：
   - 名称
   - 描述
   - 系统提示词
   - 模型选择
   - 温度参数
   - 最大Token数
4. 保存即可

### 2. 配置 AI 模型

1. 进入"模型配置"模块
2. 点击"添加模型"
3. 输入：
   - 提供商（如 OpenAI, Anthropic）
   - 模型名称
   - API 密钥
   - Base URL（可选）
4. 测试连接后保存

### 3. 备份设置

1. 点击侧边栏底部的"导出配置"按钮
   
   或

   进入"导入/导出"模块，点击"导出配置"

2. 系统会下载一个 JSON 文件，包含所有设置

### 4. 恢复设置

1. 点击侧边栏底部的"导入配置"按钮
   
   或

   进入"导入/导出"模块，点击"导入配置"

2. 选择之前导出的 JSON 文件
3. 确认后系统会恢复所有设置

### 5. 重置设置

⚠️ **危险操作**：此操作会删除所有自定义配置！

1. 点击侧边栏底部的"重置设置"按钮
2. 确认对话框
3. 系统恢复为默认设置

## 🔧 高级功能

### 自定义快捷键

进入"通用设置" → "快捷键方案"，可以：

- 选择预设方案（VSCode）
- 自定义快捷键绑定

### 代码索引配置

进入"上下文管理"：

- 配置忽略规则（如 `node_modules`）
- 设置索引深度
- 管理文档集

进入"对话流设置"：

- 选择通知方式（横幅、声音、菜单）
- 调节音量
- 配置提示音

## 💡 提示和技巧

### 1. 使用搜索快速定位

不确定某个设置在哪？使用全局搜索功能！例如：

- 搜索"主题"→ 找到主题切换设置
- 搜索"温度"→ 找到模型温度参数
- 搜索"智能体"→ 列出所有智能体

### 2. 定期备份配置

建议每周导出一次配置，保存到安全的地方。这样即使误操作也能快速恢复。

- **通用任务**：使用内置智能体

## ❓ 常见问题

### Q: 设置会自动保存吗？

**A**: 是的！所有设置修改都会立即保存到浏览器本地存储（LocalStorage），无需手动保存。

### Q: 清除浏览器缓存会丢失设置吗？

**A**: 会的。建议定期导出配置文件备份。

### Q: 可以在多台设备间同步设置吗？

**A**: 目前需要手动导出/导入 JSON 文件。未来版本会支持云端同步。

### Q: 如何查看智能体的历史版本？

**A**: 目前不支持版本控制。建议在修改前复制一份智能体。

### Q: 模型 API 密钥安全吗？

**A**: API 密钥保存在浏览器本地存储中，不会上传到服务器。但建议：
- 使用专用的 API 密钥
- 设置适当的使用额度限制
- 定期轮换密钥

### Q: 重置设置后能恢复吗？

**A**: 不能直接恢复。请在重置前导出配置备份。
